Script Reference

Exact responsibilities for the numbered scripts and standalone foundry variant.

Run all main numbered scripts from the repository root on the operator workstation. The scripts use SSH to make changes on the target systems.

Main Workflow Scripts

ScriptRuns fromChanges
01-register-rhn.shWorkstationRegisters the virtualization host and enables required repositories.
02-install-host-packages.shWorkstationInstalls virtualization, OVS, Cockpit, tmux, vim, and image tools; then reboots the host.
03-enable-host-services.shWorkstationEnables host services and the default libvirt image pool.
04-configure-ovs-networks.shWorkstationCreates persistent OVS and libvirt network setup for lab-switch.
05-verify-virt-host.shWorkstationVerifies virtualization, libvirt, OVS, services, and host state.
06-create-foundry-vm.shWorkstationCopies the RHEL cloud image, creates the foundry disk/seed ISO, and boots foundry.
07-configure-foundry-console.shWorkstationSets console passwords for appliance and cloud-user; keeps appliance passwordless sudo.
08-configure-foundry-services.shWorkstationConfigures IdM DNS, NTP, HTTP staging, firewall, podman, and image tooling on foundry.
09-verify-foundry-services.shWorkstationVerifies foundry DNS, NTP, HTTP, firewall, and upstream DNS recursion behavior.
10-prepare-appliance-assets.shWorkstationCopies local-only secrets to foundry and writes generated appliance, install, and agent config.
11-build-appliance-image.shWorkstationRuns the appliance builder container and writes appliance.raw.
12-create-cluster-config-image.shWorkstationCreates agentconfig.noarch.iso and installer state on foundry.
13-create-ocp-vms.shWorkstationTransfers artifacts to the virtualization host and creates ocp-01, ocp-02, and ocp-03.
14-destroy-ocp-vms.shWorkstationDestroys OpenShift VM domains and removes their overlay disks for reimage.
15-watch-ocp-install.shWorkstationStreams bootstrap and install wait commands through foundry.
16-verify-ocp-cluster.shWorkstationCreates a temporary API tunnel and runs sanitized oc checks.

Standalone Foundry Scripts

ScriptPurpose
